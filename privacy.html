プライバシーポリシー

最終更新日: 2025-09-10

Bookmark Vault（以下「本拡張」）は、ユーザーのブックマークを端末内のみで安全に管理することを目的としています。 本拡張はユーザーデータを外部へ送信せず、サーバーとも通信しません。

1. 収集する情報

本拡張は、個人を特定できる情報を収集しません。解析・広告・トラッキングは実施しません。

2. データの保存と暗号化

  • 保存場所: chrome.storage.local（ユーザーの端末内）
  • 暗号化: Web Crypto API を用い、ユーザーが設定した PIN から導出した鍵でデータ（タイトル・URL 等）を暗号化して保存します。
  • 送信: データを開発者または第三者へ送信・共有することはありません。

3. 要求する権限と用途

  • storage：拡張データの端末内保存に使用
  • bookmarks：Chrome 既存ブックマークの「取り込み」にのみ使用（閲覧・選択・コピー）
  • tabs：アクティブタブのタイトル/URL を取得して「このページを追加」に使用。リンクを新規タブで開く操作にも使用

4. データの保持と削除

  • データは端末内に保持され、ユーザーが設定メニューからリセットすることで削除できます。
  • 拡張のアンインストール時、ブラウザに残るデータはブラウザの仕様に従います。

5. 子どものプライバシー

本拡張は13歳未満の児童を対象としていません。

6. ポリシーの更新

内容を更新する場合は、本ページの「最終更新日」を改定します。
