# Bookmark Vault

ブラウザ標準のブックマーク機能から独立し、高度な暗号化とPINロックによって保護されたプライベート・ブックマーク管理拡張機能です。

## 概要

Bookmark Vaultは、特定のブックマークを完全に秘匿するために設計されました。データはすべて暗号化された状態でブラウザのローカルストレージ（chrome.storage.local）に保存され、パスワードを知る本人以外は内容を閲覧・復号することができないゼロ・ナレッジ設計を採用しています。

## 主な機能

### 1. 独立したブックマーク管理
* **Vault専用ストレージ**: Chrome標準のブックマークとは別に、独自にフォルダ作成、追加、削除、編集が可能です。
* **直感的な操作**: ドラッグ＆ドロップによる並べ替えやフォルダ移動、タイトル・URLのインライン編集に対応しています。
* **インポート機能**: Chromeの既存ブックマークから、必要なものだけをフォルダ単位または個別に対象を絞って取り込むことができます。

### 2. 高度なセキュリティ実装
* **強固な認証**: PINコードはPBKDF2 (SHA-256) を用いて120,000回の反復ハッシュ化を行い、ランダムなソルトと共にセキュアに保存されます。
* **データ暗号化**: 保存されるすべてのデータ（タイトル、URL、フォルダ名）は、Web Crypto APIを使用したAES-GCM方式で暗号化されます。
* **揮発性の高い鍵管理**: 復号用の鍵は `chrome.storage.session` に一時的に保持され、ポップアップを閉じた際やブラウザの再起動時に自動的に破棄されるため、再ロックが確実に行われます。

### 3. UI/UX
* **カスタマイズ性**: 各アイテムに個別のカラー設定（文字色・背景色）が可能なほか、ライト/ダークテーマの切り替えに対応しています。
* **検索機能**: 保存したデータを対象としたインクリメンタルサーチが可能です。
* **自動保護**: セキュリティ向上のため、無操作状態が一定時間続くとポップアップを自動的に閉じるタイマー機能を搭載しています。

## 技術仕様

* **プラットフォーム**: Chrome Extension Manifest V3
* **主要技術**:
    - Web Crypto API (PBKDF2, AES-GCM)
    - Chrome API (bookmarks, storage, session, tabs)
    - Vanilla JavaScript, CSS3, HTML5

## セキュリティとプライバシー

* **ローカル完結型**: ユーザーのPINやブックマークデータが外部サーバーへ送信されることは一切ありません。
* **権限の最小化**: 動作に必要な権限（bookmarks, storage, tabs等）のみを要求し、プライバシーを最大限に尊重した設計となっています。
